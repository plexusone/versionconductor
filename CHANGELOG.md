## [v0.2.0] - 2026-04-25

### Highlights

- Dependency graph analysis for multi-org module relationships
- Graph visualization in DOT (Graphviz) and Mermaid formats
- Repository moved to plexusone GitHub organization

### Added

- Dependency graph package for analyzing module relationships ([`c343cb2`](https://github.com/plexusone/versionconductor/commit/c343cb2))
- go.mod parser with support for require, replace, and exclude directives ([`c343cb2`](https://github.com/plexusone/versionconductor/commit/c343cb2))
- Graph builder to fetch and analyze modules from GitHub repos ([`c343cb2`](https://github.com/plexusone/versionconductor/commit/c343cb2))
- Topological sort for correct upgrade ordering ([`c343cb2`](https://github.com/plexusone/versionconductor/commit/c343cb2))
- Multi-org portfolio support with org and language filtering ([`c343cb2`](https://github.com/plexusone/versionconductor/commit/c343cb2))
- CLI commands: `graph build`, `graph dependents`, `graph dependencies`, `graph order`, `graph stale`, `graph stats` ([`75517d8`](https://github.com/plexusone/versionconductor/commit/75517d8))
- CLI command: `graph visualize` with DOT and Mermaid format options ([`79ea908`](https://github.com/plexusone/versionconductor/commit/79ea908))
- Caching layer with memory and file backends, TTL support ([`31b17c1`](https://github.com/plexusone/versionconductor/commit/31b17c1))
- Rate limiting with automatic retry via `mogo/net/http/retryhttp` ([`3753a0b`](https://github.com/plexusone/versionconductor/commit/3753a0b))
- Cache CLI options: `--cache`, `--cache-dir`, `--cache-ttl`, `--no-cache` ([`114ad7f`](https://github.com/plexusone/versionconductor/commit/114ad7f))
- Progress reporting for graph build operations ([`79ea908`](https://github.com/plexusone/versionconductor/commit/79ea908))

## [v0.1.0] - 2026-01-31

### Highlights

- Automated dependency PR management across multiple GitHub organizations
- Policy-based auto-review and merge of Renovate/Dependabot PRs
- Maintenance release creation with semantic version bumping

### Added

- CLI with `scan`, `review`, `merge`, `release`, and `version` commands using Cobra framework ([`1c1e14b`](https://github.com/plexusone/versionconductor/commit/1c1e14b))
- GitHub collector for dependency PR discovery across organizations ([`1c1e14b`](https://github.com/plexusone/versionconductor/commit/1c1e14b))
- Detection of Renovate and Dependabot PRs by author identification ([`1c1e14b`](https://github.com/plexusone/versionconductor/commit/1c1e14b))
- PR metadata extraction including dependency name, versions, and update type ([`1c1e14b`](https://github.com/plexusone/versionconductor/commit/1c1e14b))
- Cedar policy engine for auto-merge decisions ([`1c1e14b`](https://github.com/plexusone/versionconductor/commit/1c1e14b))
- Three built-in merge profiles: `aggressive`, `balanced`, `conservative` ([`1c1e14b`](https://github.com/plexusone/versionconductor/commit/1c1e14b))
- PR merger with squash, merge, and rebase strategies ([`1c1e14b`](https://github.com/plexusone/versionconductor/commit/1c1e14b))
- Release creator with semantic version bumping (patch releases) ([`1c1e14b`](https://github.com/plexusone/versionconductor/commit/1c1e14b))
- JSON and Markdown report formats ([`1c1e14b`](https://github.com/plexusone/versionconductor/commit/1c1e14b))
- Dry-run mode for all write operations ([`1c1e14b`](https://github.com/plexusone/versionconductor/commit/1c1e14b))
- Example Cedar policy for auto-merge patch updates ([`6646bb9`](https://github.com/plexusone/versionconductor/commit/6646bb9))
